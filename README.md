# ToDo List Web App

A simple ToDo List web application built using HTML, CSS, and JavaScript. This project helps users keep track of their daily tasks in an easy and organized way.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Search for tasks
- Dark/Light mode
- Shows total, completed, and pending tasks
- Saves tasks using Local Storage
- Delete all completed tasks
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript

## How to Run

1. Clone this repository.

```bash
git clone https://github.com/naruto15243/ToDo-List-Web-App.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

That's it! No additional setup is required.
